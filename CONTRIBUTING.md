# Contributing

## Development

1. Edit `check-installs-tui`.
2. Run it locally in a terminal.
3. Verify the UI still renders correctly in `fzf`.

## Pull Requests

- Keep changes focused.
- Update the README when behavior or install steps change.
- If the Homebrew formula needs updates, update the tap repo as well.

## Releases

1. Commit changes.
2. Tag a new version such as `v0.1.1`.
3. Push the tag.
4. Update the Homebrew formula tarball URL and `sha256` if needed.
